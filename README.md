# 智能提词器

一个专业的网页版提词工具，支持多种文本预设和快捷键控制。

## 特性

- 🎯 多种文本预设
- ⌨️ 可自定义的键盘映射
- 🎨 现代化的深色主题界面
- 📱 响应式设计，支持各种设备
- ⚡ 快速编辑功能
- 🔧 全局设置（字体大小、颜色、间距等）

## 使用方法

### 在线使用（推荐）

访问我们的在线版本：[智能提词器](#) (需要部署后更新链接)

### 本地运行

如果您想在本地运行此应用：

1. 确保您已安装 [Node.js](https://nodejs.org/) (版本 18.0.0 或更高)
2. 下载此项目的 zip 文件并解压
3. 打开命令行，进入项目目录
4. 运行以下命令：

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

5. 打开浏览器访问 http://localhost:3000

### 从构建版本运行

1. 下载最新的构建版本
2. 解压文件
3. 使用任何静态文件服务器托管 `build` 目录
   - 可以使用 `serve` 工具：
     ```bash
     npm install -g serve
     serve -s build
     ```
   - 或者使用 Python：
     ```bash
     # Python 3
     python -m http.server --directory build 8000
     ```

## 键盘快捷键

- 空格键：暂停/继续
- 上下方向键：调整滚动速度
- ESC：退出全屏
- F11：进入全屏
- 其他快捷键可在设置中自定义
